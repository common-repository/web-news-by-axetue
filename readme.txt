== Description ==

Web News By Axetue is a plugin that can be placed as a widget on any place in a blog. It displays the latest posts from Axetue, mainly which consists of social media and mobile technology analysis and tutorials.

Any questions related to this plugin or any bugs that could be reported should be sent to help@Axetue.com.

*Feature List

* customizable widget
* displays a user-defined number of links

== Installation ==

1. Extract `web-news-by-axetue.zip`.
2. Upload the `web-news-by-axetue ` folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.
